# v0.4 発動演出追加

## 追加

- カード発動後に簡易演出を表示する `pending_fx` を追加。
- 押力式：敵トークンの押し出し、壁衝突時のフラッシュと揺れ。
- 勢式：敵ヒットの拡大縮小と浮きダメージ表示。
- 熱式：熱色フラッシュと火傷表示。
- 摩擦式・質量測定・余白回収・小石生成：状態変化の浮き文字表示。
- 5式力代入の過負荷時に赤フラッシュと「過負荷！」表示。
- 撃破時は演出を少し見せてから勝利画面へ遷移。

## メモ

- まだ仮演出なので、敵の透過PNGや専用エフェクト素材を入れる前の段階。
- 次は効果音、カード選択ハイライト、敵トークンの種類別色分けを足すとさらに気持ちよくなる。
