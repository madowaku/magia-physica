# v0.6.2 BattleState抽出

## 変更内容
- `scripts/battle/BattleState.gd` を追加し、戦闘状態管理を `Main.gd` から分離しました。
- `BattleState` に以下を移動しました。
  - プレイヤーHP、式力
  - 戦闘順、戦闘番号、敵HP、壁距離
  - 火傷、小石、滑り、観測、ターン、選択中カード
  - 基本デッキ、ラン中デッキ、山札、捨て札、手札、バトルログ
  - ラン開始、戦闘開始、ドロー、手札破棄、手札からのカード除去
  - 基本コスト計算、支払い可否、次敵名の取得
- `Main.gd` は `battle_state` を参照して既存UIを描画する形に整理しました。

## 確認項目
- Godot 4.6.1 で起動し、実行ログにエラーがないことを確認しました。
- ヘッドレス smoke script で以下を確認しました。
  - 「はじめる」ボタンが会話画面へ遷移する
  - 「すぐバトル」ボタンがバトル画面へ遷移する
  - カードの「選択」が動く
  - カードの「発動」が動く
  - 「ターン終了」が動く
  - 勝利後に報酬画面が表示される
