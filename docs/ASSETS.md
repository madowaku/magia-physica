# Assets memo

画像はプロトタイプ用の参考・仮素材として整理しています。

- `images/title_screen_mockup.png` — タイトル画面モック
- `images/battle_mockup.png` — バトル画面モック
- `images/reward_mockup.png` — 勝利報酬画面モック
- `images/dialogue_mockup.png` — チュートリアル会話画面モック
- `images/card_grid.png` — カード6枚デザイン案
- `images/enemy_lineup.png` — 敵ラインナップ案
- `cards/*.png` — `card_grid.png` からの自動切り出し
- `enemies/*.png` — `enemy_lineup.png` からの自動切り出し
- `portraits/*.png` — 表情シートからの自動切り出し

Godotにインポートした後、必要に応じて `.import` ファイルが自動生成されます。
