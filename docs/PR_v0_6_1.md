# v0.6.1 UiFactory抽出

## 変更内容
- `scripts/ui/UiFactory.gd` を追加し、UI helper群を `Main.gd` から分離しました。
- 対象: `make_panel`, `add_label`, `add_label_nowrap`, `add_button`, `set_box`, `full_rect`, `add_background`, `add_texture`
- `Main.gd` は画面遷移、バトル画面の組み立て、既存ロジックの呼び出しに寄せました。
- 安全に直せる範囲で、SFX名と敵素材のローカル変数名を変更し、shadowing warning を減らしました。

## 確認項目
- Godot 4.6.1 で起動し、実行ログにエラーがないことを確認しました。
- ヘッドレス smoke script で以下を確認しました。
  - 「はじめる」ボタンが会話画面へ遷移する
  - 「すぐバトル」ボタンがバトル画面へ遷移する
  - カードの「選択」が動く
  - カードの「発動」が動く
  - 「ターン終了」が動く
  - 勝利後に報酬画面が表示される
