# マギア・フィジカ：第一式 Godot Prototype v0.2

Godot 4.x 用の小型プロトタイプです。  
`project.godot` を Godot で開き、F5 で実行できます。

## v0.2 で入ったもの

- 実際に遊べるカードバトル
- `押力式` で敵を壁に押し、壁衝突ダメージを与える
- `勢式` でダメージ、小石があると追加ダメージ
- `熱式` で火傷を付与し、ターン終了時に継続ダメージ
- `摩擦式`、`質量測定`、`余白回収`、`小石生成` の基本効果
- 山札 / 手札 / 捨て札のデッキ処理
- 1ターン3枚ドロー
- 戦闘後のカード報酬選択
- 4連戦：ぷよ魔 → 石ころゴブリン → 油まみれスライム → 黒板の番人 グラフ・ゴーレム
- 5式力代入時の簡易過負荷

## 操作

1. タイトル画面で `はじめる` または `すぐバトル`
2. バトル画面で `□=1〜5` を選ぶ
3. 手札カードの `選択` でプレビュー確認
4. `発動` でカードを使う
5. 式力が尽きたら `ターン終了`
6. 敵を倒すと報酬カードを1枚選んで次の戦闘へ

## 実装場所

- メインロジック: `scripts/Main.gd`
- カード定義: `data/cards.json`
- 敵定義: `data/enemies.json`
- 会話定義: `data/tutorial_dialogue.json`
- 素材画像: `assets/`

## 次に足すとよさそうなもの

- カードごとの演出アニメーション
- 式力をカード上の `□` にドラッグするUI
- カード強化、削除、ショップ
- 敵2体同時戦闘
- 音、効果音、画面揺れ
- 文字フォントの指定

## 注意

このプロジェクトは、まずゲームの手触りを見るためのプロトタイプです。  
画像はコンセプトアート・モックアップ素材として入れています。
